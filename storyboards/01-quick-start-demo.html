Quick Start with Demo Data

Instantly populate SeatOptima with realistic wedding data to explore all features. Perfect for demos, testing, or learning the interface without manual data entry.

4 steps ~1 min to complete Prerequisite: None
1

Open SeatOptima

Launch the application. You'll see the default empty state with the Floor Plan view. The header contains the event name, type selector, navigation tabs, and action buttons.

Click "Load Demo" Button

Explore Other Views

Use the navigation tabs to explore the demo data across all views:

  • Dashboard: See guest counts, RSVP stats, and seating progress
  • Guests: Browse 27 guests with groups, dietary info, and relationships
  • Survey: View pre-configured survey questions
  • Optimize: Run the algorithm to auto-seat remaining guests
