Seating Optimization

Run the intelligent seating algorithm to automatically assign guests to tables based on relationships, constraints, and group preferences.

4 steps ~1 min to complete Prerequisite: Tables & guests created
1

Navigate to Optimize View

Click the Optimize tab in the navigation bar. This view shows your current seating status and provides controls for the optimization algorithm.

Configure Optimization Settings

Review the optimization parameters before running. The algorithm considers:

  • Relationships: Partners, family, friends sit together
  • Avoid constraints: Keeps conflicting guests apart
  • Groups: Tries to seat group members at same table
  • Table capacity: Respects maximum seats per table

3

Run the Optimization

Click Run Optimization to start the algorithm. A progress indicator shows the optimization running. The process typically completes in a few seconds, even for large guest lists.
