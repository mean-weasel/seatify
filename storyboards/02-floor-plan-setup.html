Floor Plan Setup

Create your event's floor plan by adding tables, arranging them on the canvas, and customizing their properties like capacity, shape, and size.

6 steps ~3 min to complete Prerequisite: None
1

Navigate to Floor Plan View

Click the Floor Plan tab in the navigation bar. This view shows the interactive canvas where you'll arrange tables and guests. The collapsible sidebar on the left provides guest filtering and search.

2

Add Tables Using Toolbar

Use the toolbar buttons at the top of the canvas to add tables:

  • ⭕ Round: Circular tables (default 8 seats)
  • ▭ Rectangle: Long rectangular tables (default 10 seats)
  • ⬜ Square: Square tables (default 8 seats)

Each click adds a new table to the center of the canvas. Tables are automatically named "Table 1", "Table 2", etc.

3

Drag Tables to Position

Click and drag any table to reposition it on the canvas. Arrange tables to match your venue layout. Use Ctrl/Cmd + Scroll to zoom in/out, or Shift + Drag to pan the canvas.

4

Select a Table to Edit

Click any table to select it. A properties panel appears on the right side showing editable settings. The selected table displays a coral highlight with a dashed outline.

5

Customize Table Properties

In the properties panel, you can modify:

  • Name: Give the table a descriptive name (e.g., "Head Table", "VIP")
  • Shape: Switch between round, rectangle, or square
  • Capacity: Set seats from 1-20 using +/- buttons
  • Size: Choose Small/Medium/Large presets or set exact dimensions

6

Delete Unwanted Tables

To remove a table, select it and click the Delete Table button in the properties panel. A confirmation dialog appears if guests are assigned. You can also hover over a table to see a quick-delete (×) button.

Tip: Deleting a table automatically moves any seated guests back onto the canvas as free-floating circles, ready to be reassigned.
